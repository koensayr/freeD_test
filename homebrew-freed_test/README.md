# Homebrew Tap for FreeD Protocol Validator

This is a [Homebrew](https://brew.sh) tap for the FreeD Protocol Validator toolkit.

## How to Install

First, add this tap to your Homebrew:
```bash
brew tap koensayr/freed_test
```

Then install the package:
```bash
brew install freed-validator
```

## Available Formulas

### freed-validator
A comprehensive toolkit for validating and testing FreeD protocol data streams, including:
- Real-time packet validation
- Data logging and analysis
- Packet replay functionality
- Test pattern simulation
- Visualization tools

## Development

To install from a local checkout:
```bash
brew install --build-from-source ./Formula/freed-validator.rb
```

## Updating

To upgrade to the latest version:
```bash
brew update
brew upgrade freed-validator
```
